## 2025-05-23 - Tkinter Empty State and Cursor Patterns
**Learning:** Tkinter `tk.Listbox` requires explicit handling for empty states by inserting a disabled-looking placeholder string (e.g., using `foreground='gray'`) and updating all bound event handlers (like double-click and delete actions) to check `itemcget(idx, 'foreground') == 'gray'` to prevent processing the placeholder as actual user data. Additionally, applying the `hand2` hover cursor requires `root.option_add('*TButton*cursor', 'hand2')` for `ttk.Button` globally, and explicit `cursor='hand2'` kwarg for `tk.Listbox`.
**Action:** Apply the gray placeholder string pattern with `itemcget` validation across all `tk.Listbox` widgets to provide better context to users on empty lists, and ensure `hand2` cursors are explicitly added to interactive elements to improve click affordance in Tkinter.
