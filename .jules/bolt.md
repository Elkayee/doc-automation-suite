## 2026-05-31 - Optimize whitespace collapsing using split()
**Learning:** When optimizing whitespace collapsing (e.g. `re.sub(r'\s+', ' ', text)`), replacing it with `' '.join(text.split())` is ~5x faster. However, `.split()` collapses all whitespace, including newlines. If the original regex deliberately targeted only spaces and tabs (e.g., `re.sub(r'[ \t]+', ' ', text)`) to preserve newlines, using `.split()` will introduce a functional regression unless the target string is guaranteed to contain no newlines.
**Action:** When refactoring regex whitespace normalization, verify the exact regex pattern. Only replace `re.sub(r'[ \t]+', ' ', text)` with `.split()` if the context explicitly guarantees no newlines exist, or if converting newlines to spaces is acceptable behavior.
